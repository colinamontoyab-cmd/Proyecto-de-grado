# Sistema

## 1. Interfaz de usuario
Permitirá al estudiante escribir preguntas o ejercicios algebraicos.

## 2. Motor de IA
Procesará las preguntas utilizando modelos de lenguaje (LLM).

## 3. Sistema de prompts
Controlará el comportamiento pedagógico de la IA.

## 4. Módulo de retroalimentación
Analizará errores y explicará procedimientos paso a paso.

## 5. Base temática
Contendrá contenidos de álgebra correspondientes a grado octavo.

### Flujo general del sistema
Usuario → Interfaz → Prompt → Modelo IA → Respuesta pedagógica
