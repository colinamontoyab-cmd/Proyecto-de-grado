# Proyecto de Grado

## Objetivo
Desarrollar una Inteligencia Artificial educativa que fortalezca el aprendizaje del álgebra en estudiantes de grado octavo mediante explicaciones interactivas y apoyo personalizado.

### Descripción

Este proyecto tiene como finalidad desarrollar una herramienta educativa basada en Inteligencia Artificial que apoye a los estudiantes de grado octavo del Colegio Provinma Bilingüe en el aprendizaje del álgebra.

La aplicación estará diseñada para explicar procedimientos matemáticos paso a paso, resolver ejercicios, reforzar conceptos y brindar acompañamiento personalizado.

Busca facilitar la comprensión de temas fundamentales del álgebra escolar mediante explicaciones claras, ejemplos prácticos y retroalimentación.

### Tecnologías utilizadas

- Python.
- Visual Studio Code.
- Claude AI.
- GitHub.
- APIs (LLM).

### Funcionalidades

- Explicación paso a paso de ejercicios algebraicos.
- Resolución de ecuaciones básicas y complejas.
- Apoyo en temas de factorización.
- Simplificación de expresiones algebraicas.
- Generación de ejercicios de práctica.
- Corrección de errores en procedimientos matemáticos.
- Retroalimentación personalizada para el estudiante.
- Uso de lenguaje sencillo y adaptado.
